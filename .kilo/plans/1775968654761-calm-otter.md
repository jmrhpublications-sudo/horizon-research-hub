# Plan: Complete Rewrite of Submit Form with File Upload to Supabase Storage

## Overview
The current file upload implementation is not working properly for non-logged-in users. We need to completely rewrite the submission form to ensure PDF/DOC/DOCX files are properly uploaded to Supabase storage and saved to the database. Files must be visible in admin dashboard with preview, download, and delete functionality. Also add user submission tracking and ensure professor dashboard works fully.

## Current Issues
- Files are not being uploaded to database for non-logged-in users
- Upload mechanism is unreliable
- Admin dashboard needs enhanced file preview/download/delete features
- User submission tracking is missing
- Professor dashboard needs full functionality
