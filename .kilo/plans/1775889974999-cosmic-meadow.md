# Comprehensive Integration Plan: Google Drive as Storage System for JMRH Publications

## Overview
This plan outlines the complete integration of Google Drive as the primary storage system for user manuscript submissions in the JMRH Publications platform. The current Supabase Storage will be replaced with Google Drive for all new submissions, where each user's files will be organized in dedicated folders named after their email addresses. Administrators will be able to download submissions directly from the dashboard without requiring any special permissions.

### Key Requirements
- **File Types**: Only PDF and DOC files are allowed for upload
- **File Size Limits**: Minimum 100 KB, maximum 20 MB per file
- **Scope**: Only new submissions will use Google Drive; existing Supabase files remain unchanged
- **Access**: Admin dashboard provides download functionality with no additional permissions required
- **Cost**: Utilize Google Drive API free tier (10 GB/day quota, 1 billion requests/day)
- **User Experience**: Seamless transition for users; no changes to submission form interface

## Technical Specifications

### Google Drive Integration Requirements
- **API**: Google Drive REST API v3
- **Authentication**: OAuth 2.0 for server-side access
- **Scopes**: `https://www.googleapis.com/auth/drive.file` (per-file access)
- **Folder Structure**: `/JMRH-Submissions/{user-email}/{submission-id}/`
- **File Naming**: `{original-filename}_{timestamp}.{extension}`
- **Sharing**: Files set to private (admin access via API)

### File Validation Rules
- **Allowed Extensions**: .pdf, .doc, .docx
- **Size Range**: 100 KB - 20 MB
- **MIME Types**: application/pdf, application/msword, application/vnd.openxmlformats-officedocument.wordprocessingml.document

### Error Handling
- Network failures: Retry with exponential backoff
- API quota exceeded: Queue uploads for later retry
- Invalid files: Reject with specific error messages
- Folder creation failures: Fallback to root directory

### Phase 2: Backend Integration (3-4 days)

#### 2.1 Google Drive Service Creation
Create `src/lib/google-drive.ts`:
```typescript
import { google } from 'googleapis';

const SCOPES = ['https://www.googleapis.com/auth/drive.file'];

export class GoogleDriveService {
  private drive: any;

  constructor() {
    const auth = new google.auth.GoogleAuth({
      credentials: {
        client_email: process.env.GOOGLE_CLIENT_EMAIL,
        private_key: process.env.GOOGLE_PRIVATE_KEY?.replace(/\\n/g, '\n'),
      },
      scopes: SCOPES,
    });
    this.drive = google.drive({ version: 'v3', auth });
  }

  async createUserFolder(email: string): Promise<string> {
    // Check if folder exists
    const query = `name = '${email}' and mimeType = 'application/vnd.google-apps.folder' and trashed = false`;
    const searchResponse = await this.drive.files.list({ q: query });

    if (searchResponse.data.files && searchResponse.data.files.length > 0) {
      return searchResponse.data.files[0].id;
    }

    // Create new folder
    const folderMetadata = {
      name: email,
      mimeType: 'application/vnd.google-apps.folder',
      parents: [process.env.GOOGLE_DRIVE_ROOT_FOLDER_ID],
    };

    const folderResponse = await this.drive.files.create({
      resource: folderMetadata,
      fields: 'id',
    });

    return folderResponse.data.id;
  }

  async uploadFile(file: Buffer, filename: string, folderId: string, mimeType: string): Promise<string> {
    const fileMetadata = {
      name: filename,
      parents: [folderId],
    };

    const media = {
      mimeType: mimeType,
      body: file,
    };

    const response = await this.drive.files.create({
      resource: fileMetadata,
      media: media,
      fields: 'id,webViewLink',
    });

    return response.data.id;
  }

  async getFileDownloadUrl(fileId: string): Promise<string> {
    const response = await this.drive.files.get({
      fileId: fileId,
      fields: 'webContentLink',
    });
    return response.data.webContentLink;
  }
}
```

#### 2.2 File Validation Utilities
Create `src/lib/file-validation.ts`:
```typescript
export const ALLOWED_EXTENSIONS = ['.pdf', '.doc', '.docx'];
export const ALLOWED_MIME_TYPES = [
  'application/pdf',
  'application/msword',
  'application/vnd.openxmlformats-officedocument.wordprocessingml.document'
];
export const MIN_FILE_SIZE = 100 * 1024; // 100 KB
export const MAX_FILE_SIZE = 20 * 1024 * 1024; // 20 MB

export function validateFile(file: File): { valid: boolean; error?: string } {
  const extension = file.name.toLowerCase().substring(file.name.lastIndexOf('.'));
  if (!ALLOWED_EXTENSIONS.includes(extension)) {
    return { valid: false, error: 'Only PDF and DOC files are allowed' };
  }

  if (!ALLOWED_MIME_TYPES.includes(file.type)) {
    return { valid: false, error: 'Invalid file type' };
  }

  if (file.size < MIN_FILE_SIZE) {
    return { valid: false, error: 'File must be at least 100 KB' };
  }

  if (file.size > MAX_FILE_SIZE) {
    return { valid: false, error: 'File must be less than 20 MB' };
  }

  return { valid: true };
}
```
